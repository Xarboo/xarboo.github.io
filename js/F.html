Przykłady: function oraz const + function

Dwa przykłady: klasyczna funkcja i funkcja przypisana do stałej const. Wpisz dane i kliknij przycisk.

1. Klasyczna funkcja (function)

—
Funkcja zwykła, która zwraca kwadrat liczby.

2. Funkcja przypisana do const

—
Stała const przechowuje funkcję, która zwraca powitanie.
