Operatory JavaScript — przykłady i wyjaśnienia

Operatory arytmetyczne

Dodawanie, odejmowanie, mnożenie, dzielenie i modulo. Wynik jest liczbą. W JS używamy operatorów + - * / %.
// przykłady
let a = 10;
let b = 3;
console.log('a + b =', a + b);
console.log('a - b =', a - b);
console.log('a * b =', a * b);
console.log('a / b =', a / b);
console.log('a % b =', a % b);
console.log('++a (pre) =', ++a);
console.log('b++ (post) =', b++);
console.log('b after post++ =', b);

Operatory logiczne

AND, OR, NOT. Wartości truthy/falsy decydują o wyniku. Używamy && || !. Przydatne w warunkach i krótkim ocenie wartości.
let x = 0;
let y = 'hello';
console.log('x && y =', x && y);
console.log('x || y =', x || y);
console.log('!x =', !x);
console.log('(5 > 3) && (2 < 1) =', (5>3) && (2<1));

Operatory bitowe

Działają na reprezentacji 32-bitowej liczb całkowitych. & | ^ ~ << >> >>>.
let p = 5;  // 0101
let q = 3;  // 0011
console.log('p & q =', p & q);
console.log('p | q =', p | q);
console.log('p ^ q =', p ^ q);
console.log('~p =', ~p);
console.log('p << 1 =', p << 1);
console.log('p >> 1 =', p >> 1);
console.log('p >>> 1 =', p >>> 1);

Operatory przypisania

Proste przypisanie i skrócone kombinacje: =, +=, -=, *=, /=, %=, <<=, &=, |=, ^=. Modyfikują zmienną po lewej.
let n = 10;
n += 5; console.log('n += 5 ->', n);
n -= 3; console.log('n -= 3 ->', n);
n *= 2; console.log('n *= 2 ->', n);
n /= 4; console.log('n /= 4 ->', n);
n %= 3; console.log('n %= 3 ->', n);
let m = 1; m <<= 2; console.log('m <<= 2 ->', m);

Operatory porównania

== i ===. == konwertuje typy, === porównuje bez konwersji. != i !== analogicznie. Inne: >, <, >=, <=.
console.log("'5' == 5 ->", '5' == 5);
console.log("'5' === 5 ->", '5' === 5);
console.log('0 == false ->', 0 == false);
console.log('0 === false ->', 0 === false);
console.log('5 > 3 ->', 5 > 3);
console.log('5 <= 5 ->', 5 <= 5);

Operatory łańcuchowe i łańcuchowanie

Operacje na łańcuchach: konkatenacja (+), template literals. "Łańcuchowanie" oznacza wywoływanie metod jedna po drugiej oraz optional chaining operator ?. i nullish coalescing ??.
let s1 = 'Cześć';
let s2 = 'świat';
console.log('Konkatenacja: s1 + " " + s2 ->', s1 + ' ' + s2);
console.log('Template literal ->', `${s1}, ${s2}!`);
let person = {name:'Ana', contact:{email:'ana@ex.com'}};
console.log('Optional chaining person.contact?.email ->', person.contact?.email);
let maybe = null;
console.log('Nullish coalescing maybe ?? "domyślna" ->', maybe ?? 'domyślna');
console.log("Łańcuchowanie metod: '  ok  '.trim().toUpperCase() ->", '  ok  '.trim().toUpperCase());
