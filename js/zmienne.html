zmienne

Przykłady: var, let, const, konwersja i suma liczb

Skrypt dla Var:
var liczbaVar = 5; document.getElementById("outputVar").innerHTML = `
Przykład var: var liczbaVar = 5; => ${liczbaVar}`;
Skrypt dla Let:
let liczbaLet = 10; document.getElementById("outputLet").innerHTML = `
Przykład let: let liczbaLet = 10; => ${liczbaLet}`;
Skrypt dla Const:
const liczbaConst = 15; document.getElementById("outputConst").innerHTML = `
Przykład const: const liczbaConst = 15; => ${liczbaConst}`;
Skrypt dla Konwersji ciągu na liczbę:
let tekst = "123"; let liczba = Number(tekst); // lub parseInt(tekst), parseFloat(tekst) document.getElementById("outputConversion").innerHTML = `
Konwersja tekstu na liczbę: let liczba = Number("123"); => ${liczba}`;
Skrypt dla Sumy dwóch liczb:
let suma = liczbaVar + liczbaLet + liczbaConst + liczba; document.getElementById("outputSum").innerHTML = `
Suma wszystkich liczb: ${liczbaVar} + ${liczbaLet} + ${liczbaConst} + ${liczba} = ${suma}`;
