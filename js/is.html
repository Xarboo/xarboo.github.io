5 przykładów instrukcji sterujących w JavaScript

Każdy przykład ma krótki opis, input i przycisk — kliknij, żeby zobaczyć efekt. Trochę jak laboratorium, tylko bez probówek.

1. if / else — sprawdź wiek

—
Przykład if / else: decyzja zależna od warunku.

2. switch — dzień tygodnia

—
switch sprawdza jedną zmienną przeciw wielu wartościom.

3. for — suma liczb

—
for — powtarzanie z licznikiem (pętla). Uwaga: N może być duże — to demo.

4. while — odliczanie

—
while powtarza, dopóki warunek jest prawdziwy.

5. do...while — iteruj tablicę

—
do...while wykona ciało przynajmniej raz, nawet jeśli tablica pusta.
